# Retail Sales Performance Analysis (SQL & Tableau)

## Overview

This project analyses retail sales performance using PostgreSQL and Tableau Public to identify sales trends, evaluate profitability, and explore customer and regional performance. The project demonstrates an end-to-end data analysis workflow, from writing SQL queries for data exploration to building an interactive dashboard for business reporting. The dashboard enables users to explore key performance metrics through interactive filters and visualisations, supporting data-driven decision-making.

## Objectives

The analysis aims to answer the following business questions:

- What are the overall sales, profit, order, and customer metrics?
- Which product categories generate the highest sales?
- Which regions and states perform best?
- How do customer segments contribute to revenue?
- How do discounts affect profitability?
- What are the monthly sales trends?
- Which products generate the highest sales?

## Key Insights

- Technology generated the highest overall sales among all product categories.
- The West region recorded the highest total profit.
- California generated the largest sales revenue.
- Increasing discount levels generally reduced average profit.
- Sales peaked during the final quarter of the year, suggesting seasonal demand.
